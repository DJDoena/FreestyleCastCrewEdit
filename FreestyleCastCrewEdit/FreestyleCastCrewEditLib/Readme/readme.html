General Info

This program can be used to enter Cast and Crew data more freely and comfortably than with DVD Profiler itself.

If you want to copy, move or delete existing rows, you have to select the column before the first name (the row header).
You can select a number of rows by either using the SHIFT key or - if you want to select distinct rows - the CTRL key in combination with the mous button.
You move rows by selecting them, then press the the mouse button again and keep it pressed and move the rows upwards or downwards.
You delete rows by selecting them and then pressing the DEL key.

Technical Limitation of the Plugin

If you are using the DVD Profiler plugin - as opposed to the standalone program accessible via the Windows Start menu,
you should be aware of one technical limitation. Currently the DVDP API does not allow for adding Custom Crew Roles. This info will be lost when using the Plugin.
However, if you're using the standalone program and copy&paste the data into the edit window of DVDP, the Custom Crew Roles will also be transferred.
